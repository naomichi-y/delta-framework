ヘルパ設定ファイルの概要

ヘルパ設定ファイルは、テンプレートで使用可能なヘルパクラスの基本動作を制御する設定ファイルです。(delta が提供するヘルパについては テンプレートで使用可能なヘルパ を参照)

ヘルパには global_helpers.yml と helpers.yml の 2 種類があり、それぞれヘルパの適用範囲が異なります。

パス 説明 必須
config/global_helpers.yml グローバル設定ファイル。アプリケーション全体に影響。 ◯
modules/{module}/config/helpers.yml 単一モジュール設定ファイル。グローバル設定ファイルの内容は継承される。  

ヘルパ設定ファイルは、アプリケーションの config ディレクトリに設置されています。

YAML Code# config/global_helpers.yml
form: # ヘルパ ID
  class: Delta_FormHelper # ヘルパクラス名
  bind: TRUE
...

ヘルパ ID はテンプレートに割り当てるヘルパの識別子になります。上記例の場合、Delta_FormHelper のインスタンスは form という識別子でテンプレートに割り当てられます。従って、テンプレートからは次のようなコードでヘルパメソッドを呼び出すことができます。

Template Code#
<?php echo $form->inputText('hello') ?>

bind 属性はヘルパのインスタンスを自動的にテンプレートに割り当てるかどうかの指定です。bind が FALSE に指定されている場合は、テンプレートが読み込まれる前に手動でインスタンスの割り当てを行う必要があります。

属性一覧

次に global_helpers.yml (helpers.yml を含む) ファイルで設定可能な値の一覧を記します。

属性構成 デフォルト値 取り得る値 必須 説明
(ヘルパ ID)     (array) FALSE テンプレートに割り当てるヘルパの識別子。
  assign   (string) FALSE テンプレートに割り当てる変数名。未指定の場合はヘルパ ID が変数名として使用される。
  bind TRUE (bool) FALSE ヘルパインスタンスを自動的にテンプレートに割り当てるかどうかの指定。
  class   (string) TRUE ヘルパクラス名。

ヘルパによって指定可能な属性は異なります。 詳しくはヘルパクラスのリファレンスを参照して下さい。
