Hacks

delta はデフォルトでファイルベースのセッション管理を行いますが、memcached や データベースといった高速なストレージへハンドリングを変更することが可能です。セッションハンドラの宣言は、application.yml の session 属性で設定します。ここでは delta にあらかじめ用意されている Delta_APCSessionHandler を利用してみましょう。

YAML Code# config/application.yml
...
session:
  autoStart: TRUE

  # セッションハンドラの宣言。ここではハンドラクラスとして Delta_APCSessionHandler を利用するよう宣言
  handler: Delta_APCSessionHandler

APCSessionHandler を利用するには、実行環境に APC がインストールされている必要があります。詳しくは APC のインストールサイトを参照して下さい。APC は PECL から入手可能です。

以上の設定で、生成されるセッションは APC にハンドリングされるようになります。
