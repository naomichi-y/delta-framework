使用可能なヘルパ変数

テンプレートファイルでは次に示すヘルパ変数が使用可能です。ヘルパ変数にはあらかじめ、ヘルパクラスのインスタンスが割り当てられています。

　テンプレート変数名 ヘルパクラス 説明
$date Delta_DateHelper 様々な形式で日付を表示する。
$form Delta_FormHelper フォームを生成する。
$html Delta_HTMLHelper リンクやメッセージボックスの生成を行う。
$pager Delta_PagerHelper ページネーション機能を提供。
$request Delta_RequestHelper リクエスト情報を取得する。
$site Delta_SIteHelper サイトファイルからパラメータを取得する。
$user Delta_UserHelper ユーザ情報を取得する。

テンプレートに割り当てる変数名は変更することができます。詳しくは 設定ファイル - global_helpers.yml / helpers.yml を参照して下さい。

ヘルパメソッドの使用例

ヘルパはテンプレートエンジンに依存せずに使用することができます。

Template Code: Delta_BaseRenderer 使用時
<p><?php echo $form->inputText('nickname'); ?></p>
<p><?php echo $html->anchor('Login', 'manager/loginForm'); ?></p>
<p><?php echo $user->get('nickname'); ?></p>
Template Code: Delta_SmartyRenderer 使用時
<p><input type="text" name="nickname" value="{$form->get('nickname')}"></p>
<p>{$html->anchor('Login', 'manager/loginForm')}</p>
<p>{$user->get('nickname')}</p>
