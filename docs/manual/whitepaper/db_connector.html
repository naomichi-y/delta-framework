データベースへの接続

delta はデータベースとの接続レイヤにデフォルトで PDO を利用します。ここでは最も簡単な接続設定の方法と、実際のサンプルコードを紹介します。

初めにデータベースへの接続情報を設定ファイル (application.yml) に定義しましょう。

YAML Code# config/application.yml
database:
  default:
    dsn: 'mysql:host=localhost; dbname=delta'
    user: foo
    password: bar

"database" 下の "default" 属性 (データソース) は、接続先のデータベース情報になります。データソースは複数定義することができ、プログラム側から参照するデータベースを動的に切り換えることが可能です。

属性構成 デフォルト値 必須 説明
database {データソース名} (+)     データソース名。
  dsn   ◯ DSN 情報。
user   ◯ 接続ユーザ名。
password   ◯ 接続パスワード。
options   接続オプション。設定可能な値は PDO::__construct() の $driver_options を参照
    ATTR_PERSISTENT ◯ 永続接続モード。

表内の (+) は、対象属性が複数回指定可能なことを表します。

実際の接続コードは次のようになります。

PHP Code#
// "default" データソースを参照
$database = $this->getDatabase();

$conn = $database->getConnection();
$resultSet = $conn->rawQuery('SELECT * FROM DUAL');
$data = $resultSet->readAll();

// 接続先情報を配列で定義
$config = array();
$config['dsn'] = 'mysql:host=localhost; dbname=delta';
$config['user'] = 'foo';
$config['password'] = 'bar';
$config['persist'] = TRUE

$conn = $database->getConnectionWithConfig($config);
$resultSet = $conn->rawQuery('SELECT * FROM DUAL');
$data = $resultSet->readAll();

Delta_DatabaseManager::getConnection() メソッドの引数にデータソースを指定することで、対応するデータベースへの接続オブジェクトを返します。引数を渡していない場合は "default" データソースが参照されます。
