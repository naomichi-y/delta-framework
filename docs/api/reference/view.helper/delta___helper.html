Abstract class Delta_Helper

Description | Constants | Properties | Methods

Description

全てのヘルパの基底となるクラスです。 ヘルパのインスタンスは Delta_HelperManager::getHelper() から取得して下さい。

global_helpers.yml の設定例:

{ヘルパ ID}: # 実装クラス名。 class: # テンプレートに割り当てるヘルパインスタンスの変数名。 # 未指定の場合はヘルパ ID が変数名として使用される。 assign: # テンプレート変数の自動割り当てを行うかどうかの指定。 # TRUE を指定した場合は出力テンプレート決定時に自動的にインスタンスが割り当てられる。 bind: TRUE
