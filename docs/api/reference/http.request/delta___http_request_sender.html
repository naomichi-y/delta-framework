setRequestPath()
public Delta_HttpRequestSender setRequestPath($requestPath, $entityDecode = FALSE)

setBaseURI() メソッドで指定した基底 URI に対するリクエストパスを指定します。 例えば setBaseURI('http://localhost/') が指定された状態で setRequestPath('manager/login.do') を指定した場合、実際に発行されるリクエスト URI は 'http://localhost/manager/login.do' となります。 このメソッドで指定したパスは、send() メソッドがコールされた時点で自動的に破棄されます。

Property Type Description
$request string

リクエストパス。

$entityDecode bool

setBaseURI() メソッドを参照。

{return} Delta_HttpRequestSender

Delta_HttpRequestSender オブジェクトを返します。
