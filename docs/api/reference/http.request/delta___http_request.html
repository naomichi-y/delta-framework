Class Delta_HttpRequest

Description | Constants | Properties | Methods

Description

クライアントから要求された HTTP リクエストを管理します。

このクラスは 'request' コンポーネントとして DI コンテナに登録されているため、Delta_DIContainer::getComponent()、あるいは Delta_WebApplication::getRequest() からインスタンスを取得することができます。
