Class Delta_HttpResponse

Description | Constants | Properties | Methods

Description

クライアントに応答する HTTP レスポンスを制御します。
このクラスは 'response' コンポーネントとして DI コンテナに登録されているため、Delta_DIContainer::getComponent()、あるいは Delta_DIController::getResponse() からインスタンスを取得することができます。
Delta_HttpResponse が提供するいくつかのメソッドは、flush() メソッドによってクライアントにレスポンスが返されるタイミングで実行されます。
  • setStatus()
  • setHeader()
  • setCacheExpire()
  • sendRedirectAction()
  • sendRedirect()
  • addCookie()
  • clearCookie()
  • setContentType()
  • write()
  • writeBinary()
  • writeJSON()
  • setDownloadData()
例えば setHeader() メソッドはクライアントに HTTP ヘッダを送信しますが、setHeader() の直後で exit() をコールした場合、ヘッダはクライアントに送信されないことになります。

addCookie()
public void addCookie($name, $value, $expire = NULL, $path = NULL, $domain = NULL, $secure = FALSE, $httpOnly = TRUE)
クライアントに送信する Cookie を設定します。 実際に Cookie が送信されるのは、flush() メソッドが実行されたタイミングであることに注意して下さい。
Property Type Description
$name string
クライアントに送信する Cookie の名前。
$value string
クライアントに送信する Cookie の値。
$expire int
Cookie の有効期限を 1970 年 1 月 1 日からの経過秒数で指定。 未指定時はクライアントを閉じるまでが有効期間となる。
$path string
Cookie を有効とするサーバ上のパス。
$domain string
Cookie を有効とするドメイン名。
$secure bool
セキュア Cookie の設定。TRUE を指定した場合、セキュアな通信 (HTTPS) が行われている場合のみ Cookie を送信する。
$httpOnly bool
TRUE を指定した場合、JavaScript (document.cookie) から Cookie を参照できないようにする。 ただし、httpOnly は Internet Explorer の独自拡張のため、全ての XSS から盗聴を防げる訳ではない。
  • Throws:
    InvalidArgumentException Cookie の有効期限の指定が不正な場合に発生。
  • Author:
    Naomichi Yamakita <naomichi.y@delta-framework.org>

sendRedirect()
public void sendRedirect($uri, $status = 302)
指定した URI にリダイレクトします。 実際にリダイレクトされるのは、flush() メソッドが実行されたタイミングであることに注意して下さい。
Property Type Description
$uri string
リダイレクト先の URI。 '/' (スラッシュ) から始まるパスは相対パスとみなし、アプリケーションの URI が補完されます。 ただし将来的に URI の設計が変更されることを想定した場合、相対パスによる指定を行うべきではありません。 代わりに sendRedirectAction() メソッドを使用するべきです。
$status int
送信するレスポンスコード。301〜307 を指定可能。
  • Throws:
    Delta_SecurityException URI に CrLf インジェクションとなりうる文字列が含まれる場合に発生。
  • Throws:
    Delta_UnsupportedException サポートされていないステータスコードが指定された場合に発生。
  • Author:
    Naomichi Yamakita <naomichi.y@delta-framework.org>

public void setCacheExpire()
