Class Delta_ParameterHolder

Description | Constants | Properties | Methods

Description

配列をオブジェクトとしてアクセスするためのユーティリティクラスです。

$array = array( 'database' => array( 'default' => array( 'dsn' => 'mysql:host=localhost; dbname=delta; port=3306', 'user' => 'webapp', 'password' => '...' ) ) ); $holder = new Delta_ParameterHolder($array); // 対象要素が配列の場合、get() メソッドは新しい Delta_ParameterHolder のインスタンスを返す $databaseHolder = $holder->get('database'); // オブジェクトを配列に変換 $databaseHolder->toArray(); // '.' 区切りで要素にアクセスすることも可能 // 第二引数には値が NULL の場合に返す代替値を指定することもできる $databaseHolder->get('default.dsn', 'default'); // メソッドチェーン形式のアクセス $holder->get('database')->get('default')->get('dsn'); // 配列形式のアクセス $databaseHolder['default']; // foreach() で全ての値を取得する foreach ($databaseConfig as $name => $value) { ... } // 要素数を取得する $databaseHolder->count();
