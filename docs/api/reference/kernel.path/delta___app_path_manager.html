buildAbsolutePath()
public static string buildAbsolutePath($basePath, $targetPath, $extension)

指定されたパスを元に絶対パスを構築します。

// '{APP_ROOT_DIR}/config/custom.yml' $manager->buildAbsolutePath('config', 'custom', 'yml');
Property Type Description
$basePath string

APP_ROOT_DIR から始まる相対パス。

$targetPath string

生成対象となるパス。 '@' から始まるパスは APP_ROOT_DIR からの相対パスと見なされます。

$extension string

targetPath に付加する拡張子。

  • targetPath に同じ拡張子が含まれる場合、extension は付加されません。
  • targetPath が絶対パスと見なされる場合は変換を行いません
{return} string

構築した絶対パスを返します。

  • Author:

    Naomichi Yamakita <naomichi.y@delta-framework.org>
