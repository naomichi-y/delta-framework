appendEachString()
public static array appendEachString(array $array, $append, $type = 3, array $options = array())

配列が持つ要素 (スカラー値) の前後に文字列を結合します。 この関数は他時点配列に対応しています。

Property Type Description
$array array

対象となる配列。

$append string

array の各要素に結合する文字列。(対象要素はスカラー値のみ)

$type int

Delta_ArrayUtils::APPEND_STRING_* 定数を指定。ビット和形式で指定することも可能。 既定値は Delta_ArrayUtils::APPEND_STRING_BEFORE|Delta_ArrayUtils::APPEND_STRING_AFTER。

$options array

文字結合オプション。

  • excludes: 結合対象外とする文字を配列形式で指定。
  • force: {FALSE} 結合開始位置に append と同じ文字がある場合の結合方法。 TRUE 指定時は文字が重複しても結合するが、FALSE 指定時は結合を行わない。
{return} array

文字列を結合した新しい配列を返します。

$array = array('foo', 'bar', 'baz', 'NULL'); $options = array('excludes' => array('NULL')); $type = Delta_ArrayUtils::APPEND_STRING_BEFORE|Delta_ArrayUtils::APPEND_STRING_AFTER; $array = Delta_ArrayUtils::appendEachString($array, '"', $type, $options); // "foo", "bar", "baz", NULL echo implode(', ', $array);
  • Author:

    Naomichi Yamakita <naomichi.y@delta-framework.org>

build()
public static array build($string, $value = NULL, array &$mergeArray = NULL)

文字列 string を元に配列を構築します。

Property Type Description
$string string

配列を構成する文字列。 文字列内に '[]'、または '.' (ピリオド) 区切りのキー名が指定された場合は連想配列として認識されます。

$string = 'foo[bar]'; Delta_ArrayUtils::build($string, 'baz'); // array('foo' => 'bar' => 'baz')) $string = 'foo.bar'; Delta_ArrayUtils::build($string, 'baz'); // array('foo' => 'bar' => 'baz'))
$value string

対象となる配列に格納する値。

&$mergeArray array

構築した配列と mergeArray に指定した配列をマージします。

{return} array

構築した配列を返します。mergeArray が指定されている場合は値を返しません。

  • Author:

    Naomichi Yamakita <naomichi.y@delta-framework.org>
