createOriginalImageFromFile()
abstract public mixed createOriginalImageFromFile($path, $type, $adjustOrientation = TRUE)

イメージファイルからイメージオブジェクト (またはリソース) を生成します。

Property Type Description
$path string

参照するイメージのパス。

$type int

イメージの形式。Delta_Image::IMAGE_TYPE_* 定数を参照。

$adjustOrientation bool

TRUE を指定した場合、JPEG イメージの方向を Exif の Orientation タグに基づいて補正します。

  • Exif モジュール がインストールされている場合のみ有効です。 モジュールがインストールされていない場合は FALSE (生の画像を読み込む) と同じ動作になります。
  • GD エンジン利用時の補足: GD は回転補正は行うものの、イメージ反転 (水平反転・垂直反転) 補正を行いません。
  • ImageMagick エンジン利用時の補足: Imagick が ImageMagick 6.3 以降でコンパイルされている場合、Exif モジュールのインストールは不要となります。
{return} mixed

ライブラリが提供するイメージオブジェクト、またはリソース型を返します。 オブジェクトの生成に失敗した場合は FALSE を返します。

  • Author:

    Naomichi Yamakita <naomichi.y@delta-framework.org>
