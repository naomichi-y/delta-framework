Class Delta_DatabaseConnection

Description | Constants | Properties | Methods

Description

データベースのコネクションオブジェクトです。 コネクションは Delta_DatabaseManager::getConnection()、あるいは Delta_DatabaseManager::getConnectionWithConfig() メソッドから生成することができます。
