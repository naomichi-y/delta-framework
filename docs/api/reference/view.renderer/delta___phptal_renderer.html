Class Delta_PHPTALRenderer

Description | Constants | Properties | Methods

Description

PHPTAL 描画クラスです。 PHPTAL を有効にするには、base_dicon.yml を次のように設定する必要があります。

base_dicon.yml の設定例:

componentNames: view: class: Delta_View constructor: - Delta_PHPTALRenderer setter: # エスケープ処理は PHPTAL で行う autoEscape: FALSE includes: - vendors/PHPTAL/PHPTAL.php

ビューに変数を設定:

$view->setAttribute('greeting', 'Hello World!');

テンプレートの実装例:

<div id="content"> <p tal:content="var"></p> </div>
